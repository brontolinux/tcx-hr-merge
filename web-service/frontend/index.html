Merge heart rate data from one TCX file into another

How it works

This tool merges heart rate data from a source TCX file into a target TCX file by matching timestamps. Perfect for when you have:

  • A TCX file with correct GPS/distance data but missing heart rate
  • Another TCX file with heart rate data but incorrect GPS/distance

Privacy: Your files are processed in memory and are never stored on our servers.
